AEAD cipher implementations for TLS.

This module provides AEAD (Authenticated Encryption with Associated Data) implementations for TLS 1.2 and TLS 1.3:

  • AES-128-GCM
  • AES-256-GCM
  • ChaCha20-Poly1305
