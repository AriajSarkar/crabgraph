P-384 (secp384r1) elliptic curve cryptography.

This module provides ECDH key exchange and ECDSA signatures using the NIST P-384 curve (also known as secp384r1).

P-384 provides higher security than P-256 and is required for some compliance scenarios (e.g., certain government standards).

Requires: tls feature flag
