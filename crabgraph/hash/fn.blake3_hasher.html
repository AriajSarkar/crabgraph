Function blake3_hasher

pub fn blake3_hasher() -> Hasher
Expand description

Creates a BLAKE3 hasher for incremental hashing.

This is useful when you want to hash data incrementally (streaming) rather than all at once. BLAKE3 can take advantage of multi-threading for large inputs.

Requires: extended-hashes feature flag

§Example

ⓘ
use crabgraph::hash::blake3_hasher;

let mut hasher = blake3_hasher();
hasher.update(b"hello ");
hasher.update(b"world");
let digest = hasher.finalize();
assert_eq!(digest.as_bytes().len(), 32);
