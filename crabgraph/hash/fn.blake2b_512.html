Computes BLAKE2b-512 hash of the input data.

BLAKE2b is optimized for 64-bit platforms and provides the best performance on modern CPUs. It’s faster than SHA-512 while being cryptographically secure. BLAKE2b-512 produces 64-byte (512-bit) digests.

Requires: extended-hashes feature flag

§Performance

BLAKE2b is typically 2-3x faster than SHA-512 and is one of the fastest cryptographic hash functions available.

§Example

ⓘ
use crabgraph::hash::blake2b_512;

let digest = blake2b_512(b"hello world");
assert_eq!(digest.len(), 64);
