Expand description

AES-128-GCM cipher (128-bit key).

§Security

AES-128-GCM provides 128-bit security with authenticated encryption. Suitable for most applications. Use AES-256-GCM if you need 256-bit security.

Creates a new AES-128-GCM cipher from a 16-byte key.

§Errors

Returns error if key is not exactly 16 bytes.

§Example
use crabgraph::aead::AesGcm128;

let key = [0u8; 16];
let cipher = AesGcm128::new(&key).unwrap();
